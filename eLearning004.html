  1. You are given the function y = f(x) = − 2 x + 10.
    1. Draw the graph of the function (for x between 0 and 10). Hint: For the next exercises it might be helpful to consider a range from -10 to 30 for the y-axis!
    2. What is the slope of the function?
    3. Is the relationship between the independent and the dependent variable positive or negative? What does this mean?
    4. Add the function

    6. Calculate the crossing point!
  2. You are given the function f(x) = x3 −12 x2 + 60 x + 100.
    1. Draw the graph of the function. (Hint: Draw the graph for x between 0 and 10!)
    2. Starting from x=3:
      • Calculate the change in y if x increases by 5 units.
      • Calculate the difference quotient.
      • What is the interpretation of the difference quotient?
    3. Calculate the derivative of the function.
    4. Draw the graph of the derivative. (Hint: Use the same dimension for the x-axis as for the graph of the original function.)
      • Describe the relationship between the function and it’s derivative. (Hint: Where is the derivative positive/negative, where is it increasing/qdecreasing, where is the slope of the function positive/negative, where is the slope increasing/decreasing, etc.)
    5. Add the function g(x) = x2 − 12 x + 60 to the graph with the derivative.
